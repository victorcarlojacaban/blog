This is a very simple blog application using Laravel 5.2. 

Specfication
	For Regular User
		Login
		Register
		Create Post
		Update created post
		Delete created post
		View all post

	For Admin
		Login
		Create all post
		Update all post
		Delete all post
		View all post

Technologies used
	Laravel 5.2
	Bootstrap
	CSS
	HTML
	Sweet alert
	Readme js
	Javascript
	Behat/Mink (BDD testing)
	Composer
